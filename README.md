# SplitSmart 💸

SplitSmart is a modern expense-sharing and bill-splitting web application designed to help friends, roommates, families, and travel groups manage shared expenses effortlessly.

## 🚀 Features

- 🔐 User Login & Registration Interface
- 👥 Create and Manage Groups
- 💰 Add Shared Expenses
- 📊 Track Individual Balances
- 🤝 Smart Debt Simplification
- 💳 Settlement Tracking
- 📱 Mobile-First Responsive Design
- 🎨 Modern Glassmorphism UI
- 📈 Expense Analytics Dashboard
- 🔔 Activity & Transaction History
- 🏷️ Expense Categories (Food, Travel, Rent, Utilities, Entertainment)

## 🎯 Project Objective

Managing shared expenses can become complicated when multiple people contribute to different payments. SplitSmart simplifies this process by automatically calculating balances, reducing unnecessary transactions, and providing a clear settlement plan.
